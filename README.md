##版本声明如下
###------------------------------版本v1.0.0------------------------------
版本v1.0.0版本功能主要如下：  
    1、view跟随手势移动  
    2、view跟随手势移动并且靠边悬浮  
    3、view滑动穿透效果  
    4、滑动标题淡入淡出效果  
    5、微信摇一摇的实现  
    6、选择时间和城市的滚轮效果  
    7、圆形进度条  
    8、QQ指定临时会话  
    9、带展开收缩动画的ExpandableListView  
    10、跳转照相机拍照  

###------------------------------版本v1.0.1------------------------------
版本v1.0.1版本功能主要如下：  
    1、修复了欢迎页面无法播放视频的问题  
    2、添加view实现抛物线动画  
    3、百分比布局的使用  
    4、EventBus的使用
